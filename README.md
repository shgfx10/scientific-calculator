# Scientific Calculator

A fully functional scientific calculator built with HTML, CSS, and JavaScript.

## Features
- Basic operations — Addition, Subtraction, Multiplication, Division
- Scientific functions — sin, cos, tan, log, ln, sqrt, power
- Trigonometry — sin-1, cos-1, tan-1
- Constants — Pi, Euler's number
- Memory functions — MC, MR, M+, M-
- Degree / Radian mode toggle
- Keyboard support
- Responsive design — works on mobile and desktop

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript

## How to Use
1. Clone the repository:
```bash
   git clone https://github.com/shgfx10/scientific-calculator.git
```
2. Open `index.html` in any browser
3. Start calculating!
